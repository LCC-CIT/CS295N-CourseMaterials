## Learning Objectives for Week 9

### Authentication and Authorization with Identity

This week your will learn how to use ASP.NET Core *Identity* for *authentication* and *authorization.* In particular you will learn to set up:

- A page for registering users
- A login page for authentication
- User roles
  - Based on roles, authorize users to only access certain
  - A user manager page for use by administrators
